Estudos de Probabilidade

Este repositório reúne materiais desenvolvidos ao longo do meu estudo em um curso de Probabilidade e Estatística. Os conteúdos foram construídos com o objetivo de consolidar conceitos teóricos por meio de exemplos práticos e aplicações utilizando notebooks.

Estrutura do Repositório

Os arquivos estão organizados principalmente em notebooks que acompanham a progressão das aulas, como:

Aula_1_Probabilidade.ipynb
Aula_2_Probabilidade.ipynb
Aula_4_Probabilidade.ipynb
Aula_5_Probabilidade.ipynb
Aula_5.1_Probabilidade.ipynb

Além disso, o repositório conta com uma pasta auxiliar:

utils/
Contém notebooks de apoio com estudos de bibliotecas, conceitos de Python e ferramentas utilizadas ao longo do curso.

Novos notebooks e materiais serão adicionados conforme a evolução dos estudos.

Objetivo

O repositório tem como finalidade:

Consolidar o aprendizado em probabilidade e estatística
Servir como material de revisão
Registrar a evolução ao longo do curso
Apoiar a prática com exemplos computacionais
Conteúdo

Os notebooks abordam, entre outros temas:

Fundamentos de probabilidade
Espaço amostral e definição de eventos
Probabilidade condicional
Independência de eventos
Distribuições de probabilidade
Exercícios e aplicações práticas
Observações

Este é um repositório de caráter pessoal, voltado para estudo. Os conteúdos podem incluir:

Simplificações conceituais
Interpretações próprias
Resoluções de exercícios com abordagem prática

Portanto, não substitui materiais acadêmicos formais.

Atualizações

O repositório está em constante atualização, com a inclusão de novos notebooks, exemplos e materiais complementares conforme o andamento do curso.
